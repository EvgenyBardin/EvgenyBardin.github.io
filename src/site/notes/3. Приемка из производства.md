| 1.   | Цилиндры. Приемка из производства цилиндров одной   | I   |          |               |
|      | варки осуществляется на 5 паллет:                   |     |          |               |
|      |                                                     |     |          |               |
|      | - ОК -- корректный вес и вид.                       |     |          |               |
|      |                                                     |     |          |               |
|      | - HP -- вне границ веса 4.6-4.9 (автоматический     |     |          |               |
|      |   сброс на отдельный лоток)                         |     |          |               |
|      |                                                     |     |          |               |
|      | - HR -- неудовлетворительный внешний вид            |     |          |               |
|      |                                                     |     |          |               |
|      | - QM -- для ОКК (обычно первый цилиндр и второй при |     |          |               |
|      |   окончании варки)                                  |     |          |               |
|      |                                                     |     |          |               |
|      | - XR -- внутренние включения (автоматический сброс  |     |          |               |
|      |   на отдельный лоток)                               |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+
| 2.   | При проходе по линии происходит взвешивание и       | I   |          |               |
|      | печать этикетки из Гидры. В Гидре учитывается весь  |     |          |               |
|      | каждого блока.                                      |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+
| 3.   | По окончанию варки (или при физическом наполнении   | I   |          |               |
|      | паллеты с годным продуктом) паллеты всех типов      |     |          |               |
|      | закрываются, идет передача информации из Гидры в    |     |          |               |
|      | ERP. В ERP отрабатывает функциональный модуль       |     |          |               |
|      | ZZP_RUECKM_HK_SKG с логикой транзакции              |     |          |               |
|      | ZWM_BEL_UMLPRODFERT. Создаются документы материала  |     |          |               |
|      | и транспортный заказ (на транспортный заказ         |     |          |               |
|      | печатается этикетка). Транспортные заказы           |     |          |               |
|      | подтверждает оператор складского учета и сверяет    |     |          |               |
|      | количество с МОК.                                   |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+

|      |                                                     |     |          |               |

| 5.   | Размещение ОК: одна варка -- это 1 или 2 паллеты.   | I   |          |               |
|      | Все варки за день составляют одну линию в зоне      |     |          |               |
|      | напольного хранения. Высота колонны паллет в линии  |     |          |               |
|      | 3-4 паллеты.                                        |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+
| 6.   | Размещение HP, HR, XR: ставятсяв проход перед 50й   | I   |          |               |
|      | -57 линии, после переводятся в HRU на склад ГП или  |     |          |               |
|      | зону созревания. Далее отгружаются на РАОС.         |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+
| 7.   | Размещение QM: размещается в ближайшей линии к      | I   |          |               |
|      | Метлеру                                             |     |          |               |
|      |                                                     |     |          |               |
|      | ![](media/image4.png){width="2.333659230096238in"   |     |          |               |
|      | height="2.1252963692038493in"}                      |     |          |               |
|      |                                                     |     |          |               |
|      | Физически размещается в один поддон, по системе     |     |          |               |
|      | может быть больше (по количеству варок). По         |     |          |               |
|      | требованию накладной передаются в ОКК, по системе   |     |          |               |
|      | списывается вес использованный ОКК на 4020. Далее   |     |          |               |
|      | цилиндры переупаковываются, перепечатываются        |     |          |               |
|      | этикетки и передаются на склад созревания. При      |     |          |               |
|      | необходимости процесс передачи в ОКК и переупаковка |     |          |               |
|      | повторяется. Далее переводится в HRU и отправляется |     |          |               |
|      | в РАОС.                                             |     |          |               |
+------+-----------------------------------------------------+-----+----------+---------------+
| 8.   | Блоки: С одной варки выпускается два поддона без    | I   |          |               |
|      | деления по типу. Далее происходит перемещение в     |     |          |               |
|      | зону созревания. Через 12 часов из каждой партии    |     |          |               |
|      | (варки) изымается по одному блоку для ОКК. Часть    |     |          |               |
|      | блока, использованная ОКК, расходуется на 4020.     |     |          |               |
|      | Далее этот блок переупаковывается, этикетка на него |     |          |               |
|      | перепечатывается. Если контроль ОКК пройден партия  |     |          |               |
|      | перемещается на склад ГП.                           |     |          |               |

| 13.  | Нарезка: при движении коробов по линии на них       | I   |          |               |
|      | наклеивается групповая этикетка по требованиям      |     |          |               |
|      | Меркурия (эмблема, состав, дата производства).      |     |          |               |
|      | Печатающий принтер Videojet интегрирован с          |     |          |               |
|      | оборудованием. Шаблон для печати настраивается из   |     |          |               |
|      | SAP. Шаблон для одного артикула одинаковый в        |     |          |               |
|      | течение дня. Транзакции для настройки печати:       |     |          |               |
|      |                                                     |     |          |               |
|      | - ZPACK_PRINT - Программа печати этикеток VideoJet  |     |          |               |
|      |   (перепечать этикетки)                             |     |          |               |
|      |                                                     |     |          |               |
|      | - ZPACK - Монитор печати по линиям (ведение партий, |     |          |               |
|      |   артикулов и отправка данных на принтер)           |     |          |               |
|      |                                                     |     |          |               |
|      | - ZPACK_MAT_DATA - Ведение данных для печати        |     |          |               |
|      |   этикеток ECR (тексты)                             |     |          |               |
|      |                                                     |     |          |               |
|      | Короба снимаются и укладываются на поддон. После    |     |          |               |
|      | наполнения поддона (176 коробов), старший оператор  |     |          |               |
|      | фасовки транзакцией ZWM_BEL_UMLPRODFERT создает     |     |          |               |
|      | документ материала, транспортный заказ и печатает   |     |          |               |
|      | этикетку по транспортному заказу, этикетка          |     |          |               |
|      | вкладывается в паллет. Полный паллет выставляется в |     |          |               |
|      | коридор между складами упаковки и ГП, откуда        |     |          |               |
|      | сотрудник склада сразу же перемещает их на склад    |     |          |               |
|      | ГП. Место выбирается вручную исходя из доступных    |     |          |               |
|      | мест по размеру мест.                               |     |          |               |
